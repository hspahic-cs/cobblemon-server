# Gym trainers

Every gym leader's team — levels, abilities, held items, and moves. Each gym has a standard fight and a Hard Mode "Challenge" variant (higher levels, more held items, same Pokémon line).

Gyms 1–10 are the **mainline** progression. Gyms 11–24 are **optional** (rotating + Elite Four + Champion).
