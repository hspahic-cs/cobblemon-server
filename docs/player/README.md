# Player guide

Plain-English how-to for playing on the server.

## First time?

1. **[Install the modpack](installing.md)** in PrismLauncher
2. **[Get the server IP and connect](connecting.md)**
3. Pick a team on first login and start with gym 1

## Day-to-day

- **[Economy](economy.md)** — how coins and BP work, what to buy where
- **[Gym progression](gym-progression.md)** — beating gyms, challenge mode, rewards
- **[Ranked PvP](ranked-pvp.md)** — the ladder, betting on matches, how Elo works
- **[Teams & factions](teams-and-factions.md)** — Valor / Mystic / Instinct, sub-factions, territory, the legendary attractor
- **[/feedback](feedback.md)** — file a bug or suggestion in-game; F2 attaches a screenshot

## Need help?

If something's broken or unclear, run `/feedback bug <description>` in-game. It opens a
GitHub issue automatically — see the [feedback guide](feedback.md) for the consent flow
around screenshots.
