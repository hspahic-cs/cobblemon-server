# Connecting to the server

## You'll need

- The modpack installed in PrismLauncher — see [installing.md](installing.md)
- The server IP — **ask Harris on Discord**. We don't post it publicly to keep
  random scanners off the server.

## Joining

1. Launch the modpack instance from PrismLauncher
2. Wait for the title screen, click **Multiplayer → Add Server**
3. Server Name: anything you want (e.g. `Cobblemon`)
4. Server Address: paste the IP Harris gave you, including the port
5. **Done → Join Server**

The first connection downloads server-side resource overrides, which can
take 30–60 seconds. Subsequent connects are instant.

## What happens on first login

You're greeted with a starter kit and asked to pick a team. See
[teams-and-factions.md](teams-and-factions.md) for what each team means.

## Trouble?

If you can't connect:

- Make sure the modpack version on your client matches the server. The
  server tells you the expected version on the launcher status line.
- Try a fresh `.mrpack` import — sometimes leftover jars from a prior
  modpack version cause mod-load conflicts.
- If everything else works but the server is unreachable, ping Harris;
  the dev server is a home VM and occasionally goes down for upgrades.
