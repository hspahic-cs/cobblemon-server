# PvP rules

1. **1 Legendary, Mythical, or Paradox Pokémon maximum** per team.
2. **No Terastallizing or Dynamax.**
3. **Banlist.** These forms are not allowed in ranked. The base Pokémon is usually still fine, only
   the listed mega/primal/crowned/ultra form is banned. A team with a banned Pokémon is rejected at
   team select, so swap it out and confirm again.

| Banned | Notes |
|---|---|
| **Mega legendaries** | Any Legendary/Mythical holding a Mega Stone (e.g. Mega Mewtwo X/Y, Mega Latios/Latias, Mega Diancie). |
| **Mega Rayquaza** | Rayquaza that knows Dragon Ascent. |
| **Primal Kyogre** | Kyogre holding the Blue Orb. |
| **Primal Groudon** | Groudon holding the Red Orb. |
| **Ultra Necrozma** | Necrozma holding Ultranecrozium Z. |
| **Zacian-Crowned** | Zacian holding the Rusted Sword. |
| **Zamazenta-Crowned** | Zamazenta holding the Rusted Shield. |
| **Calyrex-Shadow** | The Shadow Rider forme. |
| **Miraidon** | Banned outright. |
