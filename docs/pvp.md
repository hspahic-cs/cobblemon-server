# PvP rules

1. **1 Legendary, Mythical, or Paradox Pokémon maximum** per team.
2. **No Terastallizing or Dynamax.**
3. **Banlist.** A few power-forms are blocked in ranked. Only the listed form is banned, not the base
   Pokémon (base Kyogre, Dusk-Mane Necrozma, Hero-form Zacian, Calyrex Ice Rider, and the like all
   stay legal). A team with a banned Pokémon is rejected at team select, so swap it out and confirm
   again.

| Banned | What triggers it |
|---|---|
| **Mega Mewtwo (X/Y)** | Mewtwo holding Mewtwonite X or Y. |
| **Mega Rayquaza** | Rayquaza that knows Dragon Ascent. |
| **Primal Kyogre** | Kyogre holding the Blue Orb. |
| **Primal Groudon** | Groudon holding the Red Orb. |
| **Ultra Necrozma** | Necrozma holding Ultranecrozium Z. |
| **Zacian-Crowned** | Zacian holding the Rusted Sword. |
| **Zamazenta-Crowned** | Zamazenta holding the Rusted Shield. |
| **Calyrex (Shadow Rider)** | The Shadow Rider forme. |
| **Miraidon** | Banned outright. |

The other Mega legendaries (Latias, Latios, Diancie) and Koraidon are allowed.
