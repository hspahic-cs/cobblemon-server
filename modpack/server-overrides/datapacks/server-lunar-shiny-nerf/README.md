# server-lunar-shiny-nerf

Reduces the Cobblemon Integrations **Blue Moon** shiny boost.

Blue Moon (Enhanced Celestials) multiplies shiny odds via `cobblemonintegrations`'
`pokemon_lunar_event` data. Defaults were **64×** (Blue Moon) / **128×** (Super Blue Moon) — far too
generous. This datapack overrides those two entries to **8×** / **16×** (exp/EV multipliers and
`min_perf_ivs` left at the mod defaults). Other lunar events (blood/harvest moon) are unchanged.
