# Regular gym battles — shared rotating pool

These play during **all** regular gym leader fights (gyms 1–19). Drop ~10–12
gym battle tracks here; one is picked at random per battle and looped. The Elite
Four uses its own per-member themes (../elite4-battle/), not this pool.

Good fits: the mainline "Battle! (Gym Leader)" / "Vs. Gym Leader" themes across
generations — e.g. Gen 4 DPPt Gym Leader, Gen 5 BW Gym Leader, Gen 9 SV Gym, the
HGSS/FRLG Gym Leader remixes, Kanto/Johto Leader theme, etc. Any format ffmpeg
reads. File names are cosmetic.

Then run: python3 ops/soundtracks/build-soundtracks.py
